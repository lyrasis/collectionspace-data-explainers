= Your CollectionSpace version and profile

Log into your CollectionSpace instance.

Look at the bottom right corner of any screen and you will see something like:

....
Release 8.3.0
UI version 10.2.1
....

The above indicates your CollectionSpace application version is 8.3.0. There is no profile listed, so you have the "core" profile.

....
Release 8.3.0
UI version 10.2.1
Anthropology profile version 9.2.0
....

The above indicates your CollectionSpace application version is 8.3.0. You have the Anthropology (anthro) profile.
